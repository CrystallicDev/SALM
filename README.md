## Description
**SALM** is a library mod used by **Cristallite**. It serves as a partial Forge port of some other Fabric libraries, like bclib and wunderlib. Corresponding code belongs to their original creator.

- [BCLib](https://github.com/quiqueck/BCLib)
- [WunderLib](https://github.com/quiqueck/WunderLib)

Feel free to integrate SALM in any of your modpacks, or create new versions of this mod. However, any redistributed version, modified or not, must stay fully free, and keep a link to the original.
